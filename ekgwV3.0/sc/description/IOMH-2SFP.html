IOMH-2SFP

2 ports SFP+ 10GbE Fiber Ethernet IOM Module for AIH Series
Features
Intel X710 10 Gigabit Ethernet controller with PCIe v3.0, x8, x4 support
2x SFP+ Fiber Connectivity 
Supports SR-IOV based virtualization
On-chip QoS and Traffic Management
Virtual Machine Device Queues (VMDq)
IEEE 1588-compliant
Support for packets up to 9.5 KB (Jumbo Frames)
Operating Temperature: -10°C ~55°C
